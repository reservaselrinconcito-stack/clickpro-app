# Branding Assets: ContikPro

Este paquete contiene los activos de branding de ContikPro listos para su uso en OptimusHub.

## Archivos Principales
- **icon.svg**: El icono "solo" (simbolo) en formato vectorial. Es la mejor opción para la mayoría de los casos de uso web por su escalabilidad.
- **icon.png**: Versión de alta resolución del icono principal (extraído de los assets de Tauri).
- **logo.svg**: Logo completo que incluye el texto de la marca.

## Otros Formatos
- **favicon.png / favicon.ico**: Para uso en la pestaña del navegador.
- **apple-touch-icon.png**: Icono optimizado para dispositivos iOS.
- **icon.icns**: Icono nativo para aplicaciones macOS.

## Origen de los archivos
- `icon.svg` -> `public/landing/brand/icon.svg`
- `logo.svg` -> `public/landing/brand/logo.svg`
- `icon.png` -> `src-tauri/icons/icon.png`
- `favicon.png` -> `public/landing/favicon.png`
- `apple-touch-icon.png` -> `public/landing/apple-touch-icon.png`
- `favicon.ico` -> `src-tauri/icons/icon.ico`
- `icon.icns` -> `src-tauri/icons/icon.icns`

## Recomendación de Uso
Para **OptimusHub (Web)**, se recomienda utilizar el archivo `icon.svg` para el listado de aplicaciones, ya que garantiza la máxima nitidez en cualquier resolución.
